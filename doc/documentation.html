• V-REP module documentation

This module creates a shared library for use by V-REP (from Coppelia Robotics). It adds haptic capability to V-REP by creating an invisible CHAI3D scene that runs concurrently to V-REP and interacts with any connected haptic device. The plugin can be controlled by from V-REP LUA scripts using the LUA functions referenced in the documentation.
