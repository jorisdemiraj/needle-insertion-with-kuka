Getting started on Mac OS X using XCode

This guide assumes that your V-REP module installation is located in the root of your CHAI3D installation (e.g. ~/chai3d/modules/V-REP)

Prerequisites

  • Make sure that XCode is installed.

Compilation

  • open the main CHAI3D-V-REP XCode project file called chai3d-V-REP.xcodeproj located in the installation root folder
  • select the All project group:
  • build chai3d-V-REP and all examples for running (Shift+Command+R)

Copying the shared library and trying the examples

    • copy the libv_repExtCHAI3D.dylib shared library (and all other files) from ~/chai3d/modules/V-REP/bin/mac-x86_64 to your V-REP base installation
    • load an example V-REP scene (e.g. ./01-V-REP-shapes.ttt) into V-REP
    • start the V-REP simulation
